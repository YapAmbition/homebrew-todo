如何使用: 
```
git clone git@github.com:YapAmbition/homebrew-todo.git
cd homebrew-todo
chmod +x install.sh
./install.sh
source ~/.bashrc
```

使用帮助
```
命令列表
todo                    列出最后3条待办
todo -h                 帮助
todo -a                 列出所有待办
todo -A "something"     新增一条内容为something的待办
todo -d                 删除最后一条待办
todo -D [num]           删除一条序号为num的待办
```
